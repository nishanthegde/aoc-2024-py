## Setup

This project uses [uv](https://docs.astral.sh/uv/) for Python environment + dependency management.

```bash
# clone the repo
git clone git@github.com:nishanthegde/aoc-2024.git
cd aoc-2024

# set up environment
uv python install 3.13
uv venv --python 3.13 .venv
uv sync
